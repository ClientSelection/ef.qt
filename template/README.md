# Usage

## Using cmake

```shell script
# generate ef.hpp
efqt generate

# create build directory
mkdir build
cd build

# build
cmake ..
make -j `nproc`

# run built demo
./efqt_starter_template
```

## Using qmake

```shell script
# generate ef.hpp
efqt generate
```

Then just open this project in Qt Creator and click `Run`.
